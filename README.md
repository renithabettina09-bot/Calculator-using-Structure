# Calculator-using-Structure

## Description
This project is a simple menu-driven calculator program written in C.
It uses structures to store input values and calculated results.
The program allows the user to perform basic arithmetic operations
such as addition, subtraction, multiplication, and division.

## Features
- Menu-driven user interface
- Uses structures in C
- Performs basic arithmetic operations
- Handles division by zero
- Beginner-friendly and easy to understand

## Technologies Used
- C Programming Language

## How to Run
1. Compile the program using a C compiler (gcc recommended)
   Example:
   gcc project.c -o project
2. Run the executable file
   ./project
